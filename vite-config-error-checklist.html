Useful evidence to gather

  • Exact command and package manager.
  • Node, Vite, framework, and plugin versions.
  • Relevant `vite.config` excerpts, not the entire private project.
  • Recent dependency or config changes.
  • First and final error lines.
  • Redaction notes for environment variables and private paths.
